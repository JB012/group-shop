# Group Shop

>Mobile app where users can store and share shopping lists and receipts. 

## About
The main purpose of GroupShop is to keep records of bought items and update the people around you on the purchases made. This is
useful for grocery shopping to prevent buying duplicate items or seeing what you recently bought. It's also a useful tool to manage
finances via recorded receipts. 

## Getting Started

This build is currently in development and only works for mobile platforms (mainly Android).

### Prerequisites

Install the following:

* [Node.js](https://nodejs.org/en/download)
* [Expo Go](https://expo.dev/go)

### Installing

Enter the following commands in a terminal to run the development build.

```
git clone 
```
```
cd group-shop/group-shop
```
```
npm i
```
```
npm start
```

Once you see a QR code displayed in the terminal, open Expo Go on your phone to scan it and
you'll be able to access the development build.

## Built With

* [Expo](https://expo.dev/) - Mobile framework used
* [Clerk](https://clerk.com/) - Integrated for user authentication
* [Supabase](https://supabase.com/) - Used for realtime database changes for messaging
